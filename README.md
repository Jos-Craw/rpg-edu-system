# 🛡️ RPG Education System: Journal
Система мониторинга успеваемости студентов, стилизованная под RPG-механики. Проект разработан как часть магистерской диссертации.

## 🚀 Основные возможности
* **Прокачка персонажа:** Опыт (XP) за домашние и классные работы.
* **Система уровней:** Автоматический расчет прогресса и уровней.
* **Визуализация данных:** Динамические графики успеваемости.

## 🛠 Стек технологий
* **Backend:** Python (Django)
* **Frontend:** HTML/CSS/JavaScript
* **Data Visualization:** Plotly / Dash (в процессе разработки)
